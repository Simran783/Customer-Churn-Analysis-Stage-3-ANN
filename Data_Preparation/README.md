# Data Preparation

This folder contains all data-related files used in Stage 3 of the Customer Churn Prediction project.

## Contents
- Dataset_ATS_v2.csv: Cleaned and preprocessed dataset used for model training and evaluation.
- preprocessor.pkl: Saved preprocessing pipeline including scaling and encoding transformations.
- target_label_encoder.pkl: Label encoder used to convert categorical churn labels into numerical form.

## Purpose
These files ensure data consistency and reproducibility across model training and prediction phases.
