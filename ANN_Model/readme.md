# ANN Model

This folder contains the Artificial Neural Network (ANN) model developed for customer churn prediction.

## Contents
- ann_churn_model.h5: Trained ANN model saved using Keras/TensorFlow.
- ANN_Architecture.pdf: Visual representation of the model architecture including input, hidden, and output layers.

## Purpose
The ANN model is designed to learn complex patterns in customer behaviour and predict churn probability.
